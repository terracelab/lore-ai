# `lore chat`

> **상태**: v0.2 예정. v0 에서는 `lore synthesize <category>` 의 프롬프트를 Claude Code 에 직접 붙여넣어 사용하세요.

L1 + L2 + L3 전체를 시스템 프롬프트로 로드하고 [prompt caching](https://docs.claude.com/en/docs/build-with-claude/prompt-caching) 을 활성화한 로컬 REPL 입니다.

```bash
lore chat --provider anthropic
```

## 설계

- 시스템 프롬프트: `.lore/DOMAIN_MAP.md` + 모든 `.lore/flows/*.md`
- 사용자 메시지마다 캐시 hit → 토큰 비용 ~10% 수준
- API key: 환경 변수 (`ANTHROPIC_API_KEY` 기본)

## 대안 (지금 사용 가능)

- Claude Code 의 MCP 통합 → [integrations/mcp](/integrations/mcp)
- 브라우저에서 Q&A → Lore Board 의 RAG 챗봇
