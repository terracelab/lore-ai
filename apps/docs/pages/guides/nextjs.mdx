# Next.js

```bash
pnpm add -D lore-ai
pnpm exec lore init --template nextjs
```

## include 패턴

```yaml
projects:
  app:
    root: ./
    language: typescript
    include:
      - 'app/**/*.{ts,tsx}'
      - 'src/**/*.{ts,tsx}'
      - 'lib/**/*.ts'
      - 'hooks/**/*.ts'
    exclude:
      - '**/*.test.*'
      - '.next/**'
      - 'node_modules/**'
```

## Vercel / Turborepo

Lore AI 자체는 Next.js 빌드와 분리됩니다. CI 에서:

```yaml
- run: pnpm install --frozen-lockfile
- run: pnpm exec lore sync
- run: git diff --exit-code .lore/
```

drift check 만 통과하면 됩니다.
