# 설치

## 요구 사항

- Node.js **18.18+** (권장 20.x)
- pnpm 9+ 또는 npm 10+
- 운영체제: macOS · Linux · Windows (WSL 권장)

## 1. 전역 설치 (권장)

```bash
npm i -g lore-ai
```

설치 확인:

```bash
lore --version
```

## 2. 프로젝트 dev dependency

각 레포가 자체 버전을 핀하고 싶을 때:

```bash
pnpm add -D lore-ai
# 사용:
pnpm exec lore init
```

## 3. 일회성 실행 (`npx`)

```bash
npx lore-ai init
```
